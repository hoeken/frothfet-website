# Software

FrothFet firmware is fully open source and licensed under the **GPLv3** license.

View the source on [github](https://github.com/hoeken/frothfet-firmware).

## Firmware Structure

The firmware uses the [YarrboardFramework](https://github.com/hoeken/YarrboardFramework) under the hood:
- Hardware dimming with PWM
- Interrupt based overcurrent protection
- JSON configuration/validation system
- Multiple API options
